# Draft Abstract

Agent evaluations commonly score final outputs while leaving purpose preservation, authorization and intermediate evidence handling under-specified. We introduce DIKWP-PACT, an offline paired-control benchmark and trace specification for purpose-aware agent assurance. PACT-Bench contains 72 synthetic scenarios spanning health communication, tourism, public service, enterprise operations, research collaboration and emergency coordination. Each risk case is paired with a control that preserves the task while changing one causal variable. The benchmark measures decision accuracy, redline recall, benign-control allowance, purpose-drift detection, permission integrity, evidence closure, audit-ledger integrity, trace completeness and recovery quality. A deterministic reference implementation demonstrates the evaluation pipeline without model APIs. DIKWP-PACT does not claim to measure subjective consciousness; it operationalizes action-relevant purpose and cognitive-process integrity for auditable agents.
