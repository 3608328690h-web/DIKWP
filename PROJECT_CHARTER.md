# PROJECT CHARTER / 项目章程

## Mission

把 DIKWP 的差异化优势压缩为一个国际同行能够下载、运行、反驳、扩展和引用的旗舰项目：开放意图保障与认知轨迹基准。

## Non-goals

- 不证明现象意识或主观体验；
- 不为任何模型提供“可信认证”；
- 不连接真实医疗、政务、金融、应急或企业生产系统；
- 不建立不可申诉的个人、组织或国家排名；
- 不用仓库数量代替科学影响。

## Proposed leadership

- Project Principal / 项目负责人：Yucong Duan（公开前确认）
- Benchmark Lead：负责场景、配对对照、统计和泄漏控制
- Runtime Lead：负责规范、评分器、CI 和复现包
- Governance Lead：负责目的合同、利益冲突、版本和申诉
- External Replication Lead：负责盲测、复现报告和外部维护者

## 90-day definition of done

1. 发布可引用的 v0.1.0 Release，而不是只上传 ZIP；
2. 72 个场景、4 个基线、全部测试和哈希可复现；
3. 至少两份独立复现报告；
4. 至少一名非创始 maintainer 获得合并权限；
5. 发布预注册和英文摘要；
6. 公开一次失败、争议或评分修订；
7. 形成一个海南/WCAC 合成场景试点，不接触真实个人数据。
