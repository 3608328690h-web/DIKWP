# DIKWP-PACT v0.1.0: Project Brief

DIKWP-PACT is an open, offline-first benchmark and reference runtime for purpose-aware agent assurance. It asks a narrow, falsifiable question: **does an agent preserve an explicitly signed purpose and permission boundary while transforming data into information, knowledge, judgment and action?**

The release contains 72 paired synthetic scenarios, four deterministic baselines, JSON schemas, an offline dashboard, a preregistered experiment plan, governance files and a GitHub Project import board.

The intended public contribution is not a claim that an AI system is conscious. It is an auditable method for testing action-relevant cognitive and purpose integrity.
